### Hakkında

- Kotlin ile geliştirilmiş bir android alışveriş uygulamasıdır.
- MVVM mimarisi kullanılmıştır.
- Retrofit ile uzak sunucudan bir api ile örnek veriler elde edilmiştir.
- Coroutines ve lifecyle live data, architectural components yapıları kullanılmıştır.
- Navigation component kullanılmıştır.
- Görseller Glide kütüphanesi kullanılarak indirilmiştir.
- Tüm görsel nesneler svg formatı ile aktarılarak xml olarak saklanmaktadır.
- Tüm string ifadeler strings klasöründe saklanmaktadır.
